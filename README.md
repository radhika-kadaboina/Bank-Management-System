# Bank Management System using Python

This project is a simple Bank Management System developed using Python and Tkinter GUI.

## Features
- Account Creation
- Secure Login using PIN
- Deposit Money
- Withdraw Money
- View Transaction History
- Check Account Balance

## Technologies and Programming Concepts
- Python
- Tkinter (GUI)
- Object-Oriented Programming (OOP)
- Dictionary for storing user data

## How the System Works
1. User creates an account by entering name, age, salary, and PIN.
2. User logs in using name and PIN.
3. After login, the user can deposit or withdraw money.
4. The system records every transaction in transaction history.
5. The user can view past transactions and logout.
